Studket Access

Secure sign-in for campus trading operations.

Management and superadmin accounts use this secure web portal. Regular users authenticate only through the app, using the API-backed flow behind the mobile client.

Open Login Open Register API Docs
Management Register staff-facing access with a named management role for moderation and operations.
Superadmin Protected registration requires a server-side invite code and shares the same hardened login flow.
User User login remains app-only and is intentionally not available through the web portal.

Set SECRET_KEY and SUPERADMIN_INVITE_CODE in `.env` for production-grade deployment.
